# PoshPet 디지털 시스템 — 통합 마스터 명세서

> **이 문서의 목적**
> 그동안 프로토타입(`index.html`)에 반영해 온 **모든 기능·비즈니스 규칙·데이터 모델·확정 요건**을 한 곳에 정리한 단일 기준 문서다.
> 향후 실제 소스코드 개발(Next.js 본 사이트)로 옮길 때, 이 문서 하나로 "데모에 무엇이 확정 구현돼 있는지"를 파악할 수 있어야 한다.
> 프로토타입은 시연·검수용 단일 HTML이고, 본 개발은 별도 스택이지만 **요건·규칙·문구·UX 결정은 동일하게 계승**한다.
>
> 상태 표기: ✅ 데모 구현 완료 / 🔷 데이터·슬롯만 준비 / ⏳ 미구현(로드맵)

## 1. 고객용 기능 (모바일 8개 영역)

### 1.1 홈 (`#home`) ✅
- 히어로: 24개월 운영·33가족·365일 배지
- 5종 예약 서비스 카드 (체중 기반 시작가 표시)
- 카페 walk-in 안내 카드 ✅
- **정기배송 진입 카드** (카페 아래, 시설 둘러보기 위) ✅ *[2026-05 추가]*
- USP 그리드 4종: 라이브 카메라 / 픽업 / 채팅 / VIP 정액권
- 통계 섹션, 시설 둘러보기 CTA
- 우상단: 비로그인 시 로그인/가입 미니 칩, 로그인 시 프로필 아바타

### 1.2 둘러보기 (`#tour`) ✅
- 좌측 세로 층 선택기 (1F~4F)
- 가로 스와이프 spot, 각 층 끝에 LIVE 카메라 카드
- 예약 가능 spot에 [안내 보기] + [예약] 버튼
- **매장 실제 사진 13장 적용 완료** ✅ *[2026-05]* — 중앙 `PHOTO_MAP`에서 관리 (아래 §7)

### 1.3 로그인 / 가입 (`openAuth`) ✅
- 4채널: Google · 전화번호(OTP 없이 즉시) · 카카오톡 · Zalo
- 채널 기반 국적 자동 추론: Kakao→KR, Zalo→VN
- 가입 직후 본인 전화번호가 "본인" 라벨 + 기본 연락처로 자동 저장
- **[R2] 로그인 vs 가입 UI 분리** ✅ — 재방문이면 "다시 오셨네요" 로그인 맥락, 신규면 가입 맥락 (문구·구성 구분)
- **[R3] 로그인 수단 기억** ✅ — 마지막 사용 수단을 "⭐ 최근 사용"으로 맨 위 크게, 나머지는 "다른 방법으로 로그인" 토글 아래 접기
  - 데모: `S._lastProvider` 상태. **실개발: localStorage 또는 서버 프로필에 마지막 provider 저장**
  - `logout()`은 마지막 수단 기억 보존, `resetAll()`(완전 초기화)은 삭제

### 1.4 펫 등록 (`openAddPet` / `renderPetSheet`) ✅
- 이름·종(강아지/고양이)·품종·체중 7단계 칩
- 백신 정보·메모, 특이사항 자유 텍스트
- 편집 모드 지원 (`d.editId`)
- **[R1] 고양이는 광견병 항목 미노출** ✅ — 고양이는 종합백신(FVRCP)만. 종 선택 시 `setPetSpecies()`가 고양이면 rabies 값 정리

### 1.5 예약 흐름 (`#flow`) ✅
- 5종 서비스: 그루밍·스파·수영장·유치원·호텔
- 단계: menu/room → date(s) → time → pet → pickup → confirm
- **[R4] 픽업 기본값 = "네, 픽업 받을게요"** ✅ + 기본 주소 자동 선택
- 저장된 주소/전화번호 기본 자동 선택, 신규 추가 시 별칭 필수
- **[R7] 픽업 주소 입력 시 "예약 확인" 버튼 활성 색 반영** ✅ (`refreshFootBtn`이 disabled + 색 동시 토글)
- **[R8] 픽업 단계 하단 입력 가림 해결** ✅ (하단 여백 확대)
- **쿠폰 자동 적용** ✅ — 적용 가능한 쿠폰이 있으면 확인 단계에서 자동 선택·차감, 변경 가능 (§5.5)
- 예약 확정 시 예약번호 PP-XXXX 발급, `paid:false`(입금 대기)
- 달력: 실제 현재 월 기준, 월 라벨 다국어, 휴무일 표시 (버그 수정 완료 §9)

### 1.6 예약 완료 (`vDone`) ✅
- 예약번호·정보 요약
- 백신 미등록 + 백신 필수 서비스면 ⚠ 안내 카드 → 펫 시트 즉시 등록
- 서비스별 방문 전 안내 (`cautionsForService`)
- 첫 예약 후 추가 정보 권유 카드 (점진적 수집)
- **첫 예약 환영 쿠폰 카드** ✅ (§5.5)
- 그루밍·스파 10회 패키지 안내 배너 ✅ (§5.4)
- 버튼: 📱 QR 결제 / 채팅 문의 / **[R5] 이 예약 수정하기** ✅ / 마이펫에서 보기

### 1.7 마이펫 (`#mypet`) ✅
- 프로필 헤더 (이름·credit 배지·정보수정)
- 빠른 액션 4장: 새 예약 / 채팅 / 정액권 / 정기 배송
- 펫 카드 (백신 배지 + 특이사항 표시)
- **내 패키지** 섹션 (활성 10회 패키지 잔여 현황) ✅
- **그루밍 주기 리마인더** 배너 ✅ (§5.4)
- **패키지 권유** 배너 ✅ (§5.4)
- **내 쿠폰** 섹션 ✅ (§5.5)
- 예약 현황: 오늘(진행/투숙 + 라이브) / 예정(D-day, 변경·취소) / **다시 이용하기**
- **[R9] 다시 이용하기**: 최대 6개, 즐겨찾기 우선→최신순, 카드 높이 축소 ✅
- 즐겨찾기 토글 (`toggleStar`)

### 1.8 더보기 (`#more`) ✅
- VIP 정액권 배너 (결제 / 선물하기)
- 직원 연결 (카톡·Zalo), 픽업 안내, 라이브 카메라, 정기 배송
- 로그아웃 (`logout()` — 수단 기억 보존)

### 1.9 예약 진입 분기 (`openServicePicker`) ✅
- **[R6]** 예약이 없으면 서비스 목록 → 신규 예약
- 예약이 1건+ 있으면 **최근 예약 1건을 상단 카드**로, 누르면 예약 현황(마이펫)으로. 그 아래 신규 예약 목록 유지

## 5. 리텐션·CRM 기능 (이탈률 45.5% 대응)

### 5.1 관리자 대시보드 강화 (Option B) ✅
- 신규→재방문 전환율 KPI
- 휴면 고객 알림 (3개월/6개월 미방문)
- VIP 잔액 총합 (회사 미래 의무)
- 월간 매출 트렌드 / 신규·이탈 추이 / 패키지 소진율 (KPI 3카드)

### 5.2 VIP 회원 사용 내역 (Option C) ✅
- `openMemberDetail`: 날짜·서비스·금액·잔액·할인·실수금 시계열
- 서비스별 필터, 기간 필터, CSV 내보내기

### 5.3 운영팀 고객 360도 뷰 ✅
- 회원 상세에 펫 프로필·백신 상태·운영 메모·고객 요약(총방문/누적/선호서비스/이용기간)·추천 액션 통합

### 5.4 그루밍 주기 추천 + 패키지 권유 ✅
- **그루밍 주기**: 개 5-6주(≈38일) / 고양이 8-10주(≈63일). 마지막 그루밍 후 주기 임박/도래 시 마이펫 리마인더 배너 + regroom 쿠폰 자동 발급
- **패키지 권유**: 그루밍/스파 2회+ 이용 & 활성 패키지 없을 때 마이펫 배너

### 5.5 쿠폰 시스템 ✅
- **첫 방문 환영 쿠폰**: 첫 예약 후 자동 발급, **다음 방문 10% 할인** (`welcome15`, 30일)
- **그루밍 정기 쿠폰**: 주기 도래 시 발급 (`regroom`, 10%, 21일, 그루밍 전용)
- 예약 확인 단계에서 적용 가능 쿠폰 자동 선택 → 차감, 사용 시 소진 처리
- 마이펫 "내 쿠폰" 섹션 노출
- ⏳ **실개발**: 쿠폰 발급/소진을 서버에서 관리, 휴면 고객 일괄 쿠폰 발송

### 5.6 백신·특이사항 시스템 (사장님 원칙) ✅
- 백신은 **펫의 고유 속성**(예약별 아님): 광견병(개만) / 종합백신(개 DHPPL·고양이 FVRCP) / 메모
- **예약은 백신 미등록이라도 확정** (마찰 최소화), 체크인 전까지 등록
- 등록 2경로: 마이펫 펫 카드 직접 등록 / 채팅으로 증명서 사진 전송
- `vaccinesSummary(v, species)`: ok/no/partial/unknown. **고양이는 광견병 제외 판정** *[R1]*

### 5.7 점진적 정보 수집 (사장님 원칙) ✅
| 단계 | 시점 | 받는 정보 | 강제성 |
|---|---|---|---|
| 가입 | 첫 진입 | 이름·연락처(채널) | 필수 |
| 펫 등록 | 첫 예약 직전 | 이름·종·품종·체중 | 필수(가격 계산) |
| 첫 예약 직후 | 자연 권유 | 펫 성별·생일·사진 / 본인 국적·성별·생일 / 마케팅 동의 | 모두 선택 |
| 이후 | 마이펫 | 위 + 백신·특이사항 | 선택 |
